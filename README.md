# QlearningSnake

**QlearningSnake** This project uses the Q-learning algorithm to design and implement a snake game. In this game, players will compete with an AI-controlled snake. The snake is controlled by an agent and makes decisions based on the state of the environment. You can switch between **AI** and **HUMAN** modes, control the snake using the keyboard, and compare scores.

## Controls
- **Y**: Start game
- **R**: Restart
- **H / TAB**: Toggle **AI ↔ HUMAN**
- **Arrow Keys**: (up down left right)key Move (Human mode)
- **- / =** (or keypad +/-): Decrease / increase speed
- **ESC**: Quit
